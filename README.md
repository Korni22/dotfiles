# Dotfiles

My dotfiles

## Install
Run `bash scripts/bootstrap`.

## Content
* `.vimrc` incl. Plugins
* Oh-my-zsh themes
* Handy zsh aliases and functions
* `.ackrc`
* `.ctags`
* `.gitconfig`
* `.tmux.conf`

## Extra
You can add a file in `~/.localrc` where you can put all your `$PATH` extensions and general local configuration.
